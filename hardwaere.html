1.1 Componentes Internos

1.1.1 CPU

A Unidade Central de Processamento (CPU) é o cérebro do computador, responsável por executar instruções e processar dados.

1.1.2 Armazenamento

Dispositivos de armazenamento, como HDDs e SSDs, são usados para salvar dados e arquivos permanentemente.

1.1.3 Memória

A memória RAM é usada para armazenar temporariamente dados que o computador está utilizando ativamente.

1.1.4 Fonte de Alimentação

A fonte de alimentação converte a energia elétrica da tomada para uma forma utilizável pelos componentes do computador.

1.2 Componentes Externos

1.2.1 Teclado

O teclado é um dispositivo de entrada que permite ao usuário digitar informações no computador.

1.2.2 Monitor de Vídeo

O monitor exibe informações visuais do computador ao usuário.

1.2.3 Mouse

O mouse é um dispositivo de entrada que permite ao usuário interagir com o computador apontando e clicando.

1.2.4 Impressora

A impressora é um dispositivo de saída que produz uma cópia física de documentos digitais.

1.2.5 Plotter

Um plotter é um dispositivo de impressão usado para produzir gráficos de grande escala, como plantas arquitetônicas.

1.2.6 Scanner

O scanner é um dispositivo de entrada que digitaliza documentos físicos para o computador.

1.2.7 Gabinete

O gabinete é a estrutura que abriga os componentes internos do computador.

1.2.8 Web Cam

A webcam é uma câmera que captura imagens e vídeos para o computador, geralmente usada para videoconferências.

1.2.9 Caixas de Som

As caixas de som são dispositivos de saída de áudio que reproduzem sons do computador.

1.2.10 Microfone

O microfone é um dispositivo de entrada que captura áudio para o computador.

1.2.11 Head Set

O headset combina fones de ouvido e microfone, permitindo comunicação de áudio bidirecional.

1.3 Interfaces de Entrada e Saída

1.3.1 Interfaces Sem Fio

Interfaces sem fio, como Wi-Fi e Bluetooth, permitem comunicação entre dispositivos sem o uso de cabos.

1.3.2 Áudio

Interfaces de áudio permitem a entrada e saída de sinais de som, como conectores de fone de ouvido e microfone.

1.3.3 Rede (Ethernet)

A interface Ethernet permite a conexão do computador a redes locais através de cabos de rede.
